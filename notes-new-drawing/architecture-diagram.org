* Shape Handling

** Domain Model (Kotlin)

#+begin_example
  Shape (data class)

** Relationships

#+begin_example
  Stylus input
    → DrawManager.newShape()
      → ShapeFactory.createShape()     (creates SDK BaseShape)
      → onShapeCompleted callback
        → EditorViewModel.addShape()   (creates domain Shape)

    Domain Shape (stored in DB) ←→ SDK BaseShape (used for rendering)

** Coordinate Systems

#+begin_example
  Three coordinate systems:

  1. NoteCoordinates      ← absolute position in the note (stored with shapes)
  2. SurfaceViewCoordinates ← position on the screen surface
  3. ScreenCoordinates    ← for pagination exclusion zones

  Transformation flow:
  Stylus input (SurfaceView coords)
    → ViewportManager.surfaceToNoteCoordinates()
      → stored as Shape in database (Note coords)

  Rendering:
  Shape from database (Note coords)
    → ViewportManager.noteToSurfaceCoordinates()
      → drawn to bitmap/canvas (SurfaceView coords)
#+end_example

** End-to-End Drawing Flow

#+begin_example
  1. Stylus touches screen
  2. TouchHelper (Onyx SDK) captures raw input
  3. OnyxStylusHandler.createOnyxCallback() receives touch points
  4. Points converted: SurfaceView coords → Note coords (ViewportManager)
  5. DrawManager.newShape() creates BaseShape via ShapeFactory
  6. onShapeCompleted() callback fires
  7. EditorViewModel.addShape() creates domain Shape
  8. NoteRepository.addShape() persists ShapeEntity to Room DB
  9. BitmapManager.renderShapeToBitmap() draws to offscreen bitmap
  10. RendererToScreenRequest enqueued to RxManager
  11. RxManager processes → bitmap blitted to SurfaceView
  12. ScreenRefreshManager triggers e-ink refresh
#+end_example

** End-to-End Erase Flow

#+begin_example
  1. Stylus in erase mode touches screen
  2. OnyxStylusHandler detects erasing
  3. EraseManager.handleErasing() called
  4. EraseManager.findIntersectingShapes() checks hit detection
  5. Matching shapes removed from ShapesManager
  6. onShapeRemoved() callback fires
  7. NoteRepository.removeShape() deletes from DB
  8. EraseManager.calculateRefreshRect() determines dirty region
  9. PartialEraseRefresh.performPartialRefresh()
  10. BitmapManager recreates bitmap without erased shapes
  11. Partial screen refresh on affected area only
#+end_example

  ├── basicRecognize()         ← returns List<RecognitionResult>

  ├── basicRecognition()           ← runs HTR and saves to RecognizedSegmentRepository

  ├── onShapesDeleted()            ← cleans up segments for deleted shapes

  RecognizedSegmentRepository
  ├── searchText(query)        ← used by HomeViewModel search
  ├── saveSegments()
  └── deleteByIds()

  Convert Selection to Text Flow:
  1. User selects strokes with SELECTOR tool (lasso)
  2. User presses "Convert to Text" button (TextFields icon) in EDIT tab
  3. EditorViewModel.convertSelectionToText() called
     → sets isConvertingToText=true (button shows spinner)
  4. HTRRunManager.recognizeShapesDirectly(noteId, selectedShapes) called
     → calls HTRManager.basicRecognize() immediately (no debounce)
     → returns merged recognized text string
  5. Original selected shapes removed from DB + SDK memory via ActionUtils
  6. New Shape(type=TEXT, points=[selectionBox.topLeft], fontSize=24f,
        fontFamily="sans-serif", strokeColor=BLACK, text=recognizedText) created
     → persisted via ActionUtils.addShapeToNoteAndMemory()
     → bitmap re-rendered, screen refreshed
  7. Selection cleared, isConvertingToText=false

  Text Drawing Flow:
  1. User taps Text tab → activates Tool.TEXT
  2. Stylus tap on canvas → OnyxStylusHandler converts to note coords
     → EditorViewModel.beginTextInput(noteX, noteY)
  3. AlertDialog shown with TextField for text entry
  4. User types text + presses OK → EditorViewModel.commitTextInput(text)
     → creates Shape(type=TEXT, points=[tapPoint], text=text,
        fontSize=current, fontFamily=current, strokeColor=textColor)
     → persists to DB, creates TextShape SDK shape, renders to bitmap
  5. Text rendered via TextShape.render() → Canvas.drawText()
     with configurable font (sans-serif/serif/monospace), size, and color
  6. Text selectable/movable/rotatable via existing selection system

  Text Formatting on Selected Shapes:
  1. User uses SELECTOR tool (lasso) to select one or more text shapes
  2. User switches to Text tab and changes font, size, or color dropdown
  3. EditorViewModel.setTextFontFamily/Size/Color() detects hasSelection
  4. Calls selectionTransformHandler.applyTextFormattingToSelection()
     → Updates in-memory TextShape SDK objects (setFontSize/Family/StrokeColor)
     → Triggers partialRefresh (or full recreateBitmap) for live update
     → Persists to DB via noteRepository.updateShape() for each selected text shape

  Universal Dropdown Close Behavior:
  - When any dropdown (text dropdowns, geometry dropdown, stroke panel) is open
    and the user touches the canvas:
    • Finger touch: OnyxDrawingActivity.setOnTouchListener checks isAnyDropdownOpen
      → calls closeAllDropdowns() which emits closeAllDropdownsEvent + resets count
    • Stylus touch: OnyxStylusHandler.onBeginRawDrawing checks isAnyDropdownOpen
      → sets isConsumingForDropdown=true, closes dropdowns, skips drawing
      → onRawDrawingTouchPointListReceived clears flag without creating a shape
  - Dropdowns register open/close with EditorViewModel.onDropdownOpened/Closed()
  - Dropdowns collect closeAllDropdownsEvent via LaunchedEffect to close themselves

  Geometry Drawing Flow:
  1. User activates ShapeButton in toolbar → Tool.GEOMETRY
  2. Stylus down → AbstractStylusHandler.beginGeometryDrawing()
     → snapshots bitmap via BitmapManager.beginGeometryDrawing()
  3. Stylus move → AbstractStylusHandler.updateGeometryPreview()
     → GeometryShapeCalculator.calculate() → note coords
     → BitmapManager.drawGeometryPreview() → restore snapshot + draw outline
  4. Stylus up → AbstractStylusHandler.finalizeGeometryShape()
     → creates BaseShape from outline points
     → adds to ShapesManager + persists via EditorViewModel.addGeometricShape()
     → onForceScreenRefresh() redraws from shapes
#+end_example
